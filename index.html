How It Works

The Process

  1. Create a source image using LaTeX (customizable via source.tex)
  2. Compile the source to PDF using pdflatex
  3. Apply the Sierpinski template pattern (8 copies in a 3√ó3 grid with center missing)
  4. Recursively rebuild each level using the previous compilation
