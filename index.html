Fractal Typography

A Sierpinski Carpet Implementation Using LaTeX

About the Project

This project implements a Sierpinski carpet using LaTeX. It first compiles a "source" image, then implements the Sierpinski template and rebuilds each level by reusing the previous compilation.

To start, install pdflatex and pdfcrop and run:

python build_recursion.py

How It Works

You can customize your own image by modifying source.tex. The Sierpinski template uses this pattern:

\newcommand{\A}{\includegraphics[width=50cm]{pdf/source.pdf}}

...

\begin{tabular}{lll}
\A & \A & \A\\
\A &    & \A\\
\A & \A & \A
\end{tabular}

A few iterations (1, 2, 3, 4, 5, and 999) can be found in the pdf directory.
